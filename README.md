# watch-nba

Watch NBA game in your terminal.

![watch-nba-gif](https://user-images.githubusercontent.com/3382565/26910744-625289a6-4c3b-11e7-95d2-6f01bfb87d17.gif)

## Install

```sh
$ npm install -g watch-nba
```
Or using yarn:

```sh
$ yarn global add watch-nba
```

## Usage

```sh
$ watch-nba
```
